	2. Por que postgresql es el rey del sql moderno ??
Explicar que es sql de forma general luego explicar postgresql y sus ventajas con un google slide

Sql (lenguaje de consulta estructurado) = es un lenguaje de programacion estandar para gestionar y manipular bases de datos relacionales (aclarar que existen tambien las bases de datos no relacionales que no se tocaran en este curso) 

Para que sirve sql:
Almacenar: información.
Buscar y recuperar: datos específicos mediante consultas  llamadas queries
Actualizar: la información existente.
Eliminar: datos de la base de datos.
Administrar: la base de datos y mejorar su rendimiento.

Entendiendo que sql es un lenguaje de programacion de consultas podemos ahora indicar que postgre es un sgbd(sistema de gestion de bases de datos relacionales)
Hace una pausa para explicar este concepto como si fuese un bibliotecario (sgbd) - biblioteca (sql) - libros (datos) 
El bibliotecario gestiona todo lo relacinado con los libros, sabe donde estan - cuantos libros tiene - te presta libros. 

usa sql como lenguaje base con algunas modificaciones en usos mas complejos,  Este es considerado el rey debido principalmente a que es  gratuito lo cual no genera costos adicionales, tambien es de codigo abierto u opensouce y por ultimo tiene una gran flexibilidad y solidez que requieren las app actuales.
